== Upgrade Notice == 
You need to do a Full Installation for this update due to changes that we do on Cron Jobs. 

1. Deactivate this plugin
2. Delete
3. Add New >> Billplz
4. Install & Activate
5. Configure this plugin
6. Save Changes (MANDATORY)
